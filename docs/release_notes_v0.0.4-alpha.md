# Release Notes - v0.0.4-alpha (2026-01-27)

本次更新标志着 epycon 正式进入**“无控制台、全 UI 控制”**时代。我们不仅重新梳理了工程结构，还对用户体验进行了一次质的飞跃。

## 核心变更点

### 1. 🚀 沉浸式后端体验 (GUI Headless)
- **取消命令行窗口**: 现在双击 `WorkMateDataCenter.exe` 后不会再弹出黑色的 CMD 框，取而代之的是静默启动并直接打开浏览器界面。
- **自动最小化逻辑**: 增强了启动阶段的窗口管理逻辑，确保用户注意力始终锁定在 UI。

### 2. 🏠 智能导航系统 (Web Navigation)
- **回显注入**: 我们不再需要手动修改每一份 HTML 文件。程序现在会自动为所有子页面（如编辑器、H5 预览）注入一个美观的“← 返回主中心”悬浮按钮。
- **UI 协同**: 主界面和子界面通过统一的样式标准实现了视觉同步。

### 3. 📝 可观测性增强 (Persistent Logging)
- **文件日志**: 虽然移除了控制台，但调试并未缺席。程序现在会将所有运行状态、转换日志记录至 `%TEMP%\epycon_gui.log`。
- **报错捕获**: 即使在无控制台环境下，系统崩溃或 IO 错误也将被完整记录。

### 4. 📁 工程结构大扫除
- ** vendor 分离**: 完成了运行时资源（Vue, Tailwind）的归位，所有三方包已迁至 `ui/vendor/`。
- ** spec 定义优化**: 打包配置现在收纳在 `scripts/` 下，实现了源码与工程配置的完美隔离。

## 升级建议
由于移除了控制台控制点，如果需要手动强制退出程序：
1. 请点击 UI 顶部的“退出”按钮（如果已配置）。
2. 或在“任务管理器”中结束 `WorkMateDataCenter.exe` 进程。
