# Release Notes - v0.0.5-alpha (2026-02-01)

本次更新是 **“黄金配置” (Golden Configuration)** 的重大里程碑，专注于将心电图 (ECG) 的信号保真度与视觉平滑度提升至临床商业级水平。

## 核心更新

### 1. 🧬 信号流水线重塑 (Signal Pipeline)
-   **谐波陷波 ActiveNotch™**: 级联处理 50Hz, 100Hz, 150Hz，在不损伤整体带宽的前提下，精准消除非线性电力干扰引起的毛刺。
-   **1阶因果滤波 (Causal)**: 恢复 40Hz 1阶低通逻辑，确保 QRS 波群无预振铃 (No Pre-ringing)，完美还原心脏电活动的物理起始点。
-   **基线稳健**: 优化了高通 (0.5Hz) 处理位置，基线更平稳。

### 2. ✨ 视觉极致优化 (Rendering Engine)
-   **微磨皮 (Micro-Smoothing)**: 引入高斯平滑层 `[0.1, 0.8, 0.1]`，优雅处理 1 阶滤波器留下的视觉底噪。
-   **4000点 LTTB**: 将降采样密度提升至 4000 点，消灭 100mm/s 高速模式下的折线感。
-   **SVG 样条曲线**: 紧致样条 (Spline 0.3) 配合原生 SVG 抗锯齿，实现真正的临床级曲线美感。

### 3. 📝 知识百科方案 (Self-Documenting)
-   **技术白皮书**: 新增 `technical_rendering_spec.md`，详细记录了这套“黄金比例”背后的数学和工程逻辑，供未来 AI 参考。
-   **Mermaid 流程图**: README 中增加了全链路处理逻辑图，一眼看清信号去向。

### 4. 🔩 系统稳定性
-   **内核补丁**: 修复了 `Channels` 核心类在处理双极导联映射时的序列化与长度检查 bug。
-   **CI 保驾护航**: 本地测试集 100% 通过 (70/70 Passed)。
